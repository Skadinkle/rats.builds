# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.0.1   | :white_check_mark: |
| 2.0.0   | :x:                |

## Reporting a Vulnerability

If there is any problems with Rv2, please head to the [Issues Tab](https://github.com/Skadinkle/rv2/issues) are report the vulnerability there.
